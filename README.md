## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Then, run the signaling server (port 4444 by default, use PORT= env varto change):

```bash
node bin/server.js
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
